# 🚀 快速启动指南

## 5分钟开始玩游戏

### 第1步：安装依赖 (1分钟)

```bash
npm install
```

### 第2步：启动游戏 (30秒)

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

### 第3步：开始游戏！ (3分钟)

1. **点击底部槽位**: 选择一个方块
2. **拖动方块**: 移到游戏区域任意位置
3. **观察效果**: 
   - 🟢 绿色边框 = 可以放置
   - 🔴 红色边框 = 无法放置
4. **松开鼠标**: 方块放置，开始下落
5. **触发消除**: 同色方块横跨左右边界时消除
6. **连锁得分**: 连续消除获得更高分数！

## 🎮 操作说明

| 操作 | 方法 |
|------|------|
| 选择方块 | 点击底部3个槽位 |
| 拖动方块 | 按住鼠标左键拖动 |
| 放置方块 | 松开鼠标左键 |
| 重新开始 | 游戏结束后点击"重新开始" |

## 💡 玩法提示

- 🎯 **策略至上**: 思考如何让同色方块连接左右边界
- 🔗 **利用物理**: 方块会像沙子一样滑落，形成三角形
- ⚡ **连锁爆发**: 消除后可能触发连锁反应，分数翻倍！
- 🎨 **颜色搭配**: 注意不同颜色的分布

## 🔧 常见问题

### Q: 游戏无法启动？
A: 确保安装了Node.js 18+，然后运行 `npm install`

### Q: 浏览器没有自动打开？
A: 手动访问 http://localhost:3000

### Q: 如何消除方块？
A: 同色方块必须同时触及左边界和右边界才能消除

### Q: 游戏结束条件是什么？
A: 当所有3个预览槽位的方块都无法放置时游戏结束
